2.アプリケーションで取り扱う利用者情報

・項目
　AAID (Android 広告 ID)
　(http://www.androiddocs.com/google/play-services/id.html)

・取得方法
　以下の広告 SDK が、内部で API を使用して取得しています。
　　・Google Mobile Ads SDK
　これ以外の部分で AAID を使用することはありません。
　設定→Google→広告→広告ID
　で、取得される ID を確認できます。

・利用目的
　広告 SDK が、AAID を広告掲載とユーザーの分析に使用します。

・同意取得の方法
　現状では同意を取得していないです。

・その他
　設定→Google→広告→広告IDをリセット
　で、AAID をリセットできます。
　リセットすることで AAID に紐づいていた情報を無効にすることができます。
　設定→Google→広告→広告のカスタマイズをオプトアウトする
　で、AAID を使用しないように設定することもできます。
　これは全てのアプリで有効な設定です。

2.User information that is treated in this application

  - User information
    - AAID (Android Advertising ID)
      (http://www.androiddocs.com/google/play-services/id.html)

  - The way to get the information
    The following advertising SDK gets the AAID by using API. 
      - Google Mobile Ads SDK
    Other modules than above don't use the AAID.
    You can confirm the AAID by Settings --> Google --> Ads --> Your advertising ID

  - Purpose of use
    It is used for advertising and user analytics.

  - Method of acquiring consent
    As of now this application doesn't obtain consent.

  - Others
    You can reset AAID by Settings --> Google --> Ads --> Reset advertising ID
    You can opt out by Settings --> Google --> Ads --> Opt out of Ads Personalization
